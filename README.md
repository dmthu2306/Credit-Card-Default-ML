# CREDIT CARD DEFAULT 

## Giới thiệu
Trong lĩnh vực tài chính, việc đánh giá rủi ro tín dụng và xác định hạn mức tín dụng phù hợp cho khách hàng là bài toán quan trọng.
Đề tài này tập trung giải quyết hai bài toán chính:
- **Bài toán dự đoán rủi ro tín dụng** (Credit Risk Prediction) : Xây dựng mô hình phân loại nhằm dự đoán khả năng một khách hàng sẽ vỡ nợ trong tương lai.
- **Bài toán đề xuất hạn mức tín dụng** (Credit Limit Recommendation) : Xây dựng mô hình hồi quy nhằm đề xuất hạn mức tín dụng phù hợp cho từng khách hàng.

## Dataset 
Đề tài sử sụng hai tập dữ liệu 
- **Dataset 1** : Dùng cho bài toán rủi ro tín dụng (classification)
- **Dataset 2** : Dùng cho bài toán đề xuất hạn mức tín dụng (regression)

## Mô hình
### Bài toán 1: Dự đoán rủi ro tín dụng
- Logistic Regression
- Random Forest 
- XGBoost

### Bài toán 2: Đề xuất hạn mức tín dụng
- Linear Regression
- Random Forest
